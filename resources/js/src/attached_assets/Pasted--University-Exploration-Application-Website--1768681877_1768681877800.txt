 University Exploration & Application Website
الهدف:
منصة عصرية واحترافية، تتيح للمستخدمين استكشاف الجامعات والكليات، الاطلاع على التفاصيل، قراءة المقالات، والحصول على توجيه ذكي حسب الاهتمامات والمؤهلات الأكاديمية، مع إمكانية التقديم للجامعات.

Contact Us Page

نموذج للتواصل:

🎨 UI / UX Requirements
professionalDesign: تصميم مهني أنيق وحديث
cleanSpacing: فراغات متقنة بين المكونات
clearTypography: خطوط واضحة وسهلة القراءة
framerMotionAnimations: حركات سلسة باستعمال Framer Motion إذا أمكن
supportLanguageSwitch: تبديل اللغة (English/Arabic) مع دعم اتجاه RTL/ LTR كامل
themeAndColorSwitcher: تبديل الألوان (فاتح/داكن + لون أساسي من عدة خيارات)
🧩 Technical Requirements
Use React + Inertia.js (الصفحات والمكونات React فقط، تتصل مستقبلًا بلارافيل)
كل المكونات تحت توقع استلام البيانات عبر Inertia props
لا يوجد منطق Backend أو Mock data
تقسيم مجلدات واضح:
/pages
/components
/layouts
/assets
/contexts
/hooks
كود نظيف وقابل للصيانة

ملاحظات هامة:

تصميم بدون لوحة تحكم أو إدارة
لا يوجد منطق خلفي
التركيز على تجربة المستخدم، الأداء، القابلية للتطوير
الالتزام بالاتجاهين (RTL/LTR)
واجهة تتوقع البيانات فقط عن طريق props (لا تعتمد على بيانات وهمية أو statically coded data)
قابلية إعادة الاستخدام وتخصيص كل المكونات
كل الصيغ واضحة ومناسبة لأي أداة AI أو مطوّر
